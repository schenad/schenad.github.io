URL命名规则

  1. Genre:

    A = “Anime/Media Mix”, D = “Dojin”, V = “Vocaloid”, C = “C-pop”, J = “J-pop”, M = “Movie”, T = “TV Drama”, H = “Heavy Metal Music”, L = “Latin Music”

  2. Related Media Work:

    Anime: 5-digit ID from MyAnimeList.net

    Media Mix: Same as Anime, using the first season

    Dojin: Related media work title in 5 digits, e. g. Touhou Project -> THP

    All other: [Empty]

  3. Original Artist (Performer&Composer):

    Anime/Media Mix: [Empty]

    All other: If performer is the composer, just write it. Otherwise, put whichever is more famous. If the name is short, put full name (e. g. AKB48, JJ-Lin), otherwise only put the initials (Hey!Say!JUMP -> HSJ).

  4. For those with multiple musical works, use the album number / single number to distinguish. For songs in the same album/single, use the track number.

  5. Instruments/Voicings:

    p = “Sheet Music for Piano Solo”, v = “Sheet Music for Piano Accompanying with Vocal”, i = “Sheet Music for Piano Accompanying Other Musical Instrument”, m = “Sheet Music for Ensemble”
